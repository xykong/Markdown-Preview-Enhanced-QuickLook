# Zoom Issues Fix Implementation Plan (Issue #27)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task.

**Goal:** 修复 Issue #27 报告的全部 6 个缩放 bug，统一缩放实现为单一系统。

**Architecture:**
- 移除 JS `transform: scale` + `width` hack，改用 `document.documentElement.style.fontSize`（CSS zoom via root font-size）触发真正的文字重排（Bug 6）
- 统一缩放入口：Swift `pageZoom` 处理原生缩放，JS 侧仅维护 font-size 覆写；两者均通过 `applyZoomLevel()` 函数入口，带 `[0.5, 3.0]` clamp（Bug 3）
- 缩放默认 **不持久化**（session-only），关闭后归 1.0（Bug 2）
- Swift `InteractiveWebView.scrollWheel` 加 `phase` 检查防止惯性误触（Bug 4）
- 新增 Reset 按钮（Bug 1），修复按钮无响应（Bug 5）

**Tech Stack:** TypeScript (Jest + jsdom), Swift (AppKit/WKWebView), CSS custom properties

## Task 2: JS GREEN — 改 setZoomLevel 为 font-size，移除 transform hack

**文件：**
- 修改: `web-renderer/src/index.ts`

- [ ] **Step 1: 修改 `window.setZoomLevel`**

找到（约第 972 行）：

```typescript
window.setZoomLevel = function(level: number) {
    currentZoomLevel = level;
    const outputDiv = document.getElementById('markdown-preview');
    if (outputDiv) {
        outputDiv.style.transform = `scale(${level})`;
        outputDiv.style.transformOrigin = 'top left';
        outputDiv.style.width = `${100 / level}%`;
    }
};
```

替换为：

```typescript
window.setZoomLevel = function(level: number) {
    const clamped = Math.max(0.5, Math.min(3.0, level));
    currentZoomLevel = clamped;
    const outputDiv = document.getElementById('markdown-preview');
    if (outputDiv) {
        // 移除旧的 transform hack（会导致布局不重排）
        outputDiv.style.transform = '';
        outputDiv.style.transformOrigin = '';
        outputDiv.style.width = '';
        // 通过 root font-size 缩放，触发真正的文字重排（修复 Bug 6）
        if (clamped === 1.0) {
            document.documentElement.style.removeProperty('--zoom-scale');
        } else {
            document.documentElement.style.setProperty('--zoom-scale', String(clamped));
        }
    }
};
```

`applyZoomLevel` 本身逻辑正确，不需要改（clamp 已在 `setZoomLevel` 中重复了，但保留双重保护无害）。

- [ ] **Step 2: 修复 scrollWheel 加 phase 检查**

找到 `InteractiveWebView` 中的 `scrollWheel` 方法：

```swift
override func scrollWheel(with event: NSEvent) {
    if event.modifierFlags.contains(.command) {
        os_log("🔵 WebView scrollWheel with CMD modifier", log: logger, type: .debug)
        
        let js = """
        (function() {
            var currentZoom = parseFloat(document.getElementById('markdown-preview')?.style.transform?.match(/scale\\(([^)]+)\\)/)?.[1] || 1.0);
            var delta = \(event.scrollingDeltaY);
            var newZoom = currentZoom + (delta > 0 ? 0.05 : -0.05);
            newZoom = Math.max(0.5, Math.min(3.0, newZoom));
            if (window.setZoomLevel) {
                window.setZoomLevel(newZoom);
            }
        })();
        """
        
        self.evaluateJavaScript(js, completionHandler: nil)
        return
    }
    super.scrollWheel(with: event)
}
```

替换为：

```swift
override func scrollWheel(with event: NSEvent) {
    // 只处理用户主动滚动（.changed phase），忽略惯性（.mayBegin/.cancelled）和 momentum（Bug 4）
    guard event.phase == .changed || event.phase == .began else {
        super.scrollWheel(with: event)
        return
    }

    if event.modifierFlags.contains(.command) {
        let delta = event.scrollingDeltaY
        guard abs(delta) > 0.1 else {
            super.scrollWheel(with: event)
            return
        }

        // 统一通过 pageZoom 缩放（与按钮行为一致）
        let newZoom = max(0.5, min(3.0, self.pageZoom + (delta > 0 ? 0.05 : -0.05)))
        self.pageZoom = newZoom
        os_log("🔵 Cmd+scroll zoom: %.2f", log: logger, type: .debug, newZoom)
        return
    }
    super.scrollWheel(with: event)
}
```
